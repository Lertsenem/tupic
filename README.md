# Tupic

Tupic is a tool to download the videos from a few websites that use RTMP.

## Requirements

* Python >= 3.2
* [RTMPdump](http://rtmpdump.mplayerhq.hu/)

## Usage

Call tupic with the full URL of the page that contains the video that you want to rip.
